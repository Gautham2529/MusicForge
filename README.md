# 🎵 MusicForge

MusicForge is a modular procedural music generation engine written entirely in Python.

It generates melodies, basslines, drum patterns, harmony and exports complete MIDI songs.

---

## ✨ Features

- 🎼 Music Theory Engine
- 🎹 Melody Generator
- 🎸 Bass Generator
- 🥁 Drum Generator
- 🎵 Chord Progressions
- 🎧 MIDI Export
- 🎼 Song Arranger
